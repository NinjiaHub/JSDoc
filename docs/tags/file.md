## <span id="overview">概述</span>

`@file` 标签提供文件的描述信息。将包含该标签的 JSDoc 注释放在文件顶部。

## <span id="examples">示例</span>

**文件描述**

```javascript
/**
 * @file Manages the configuration settings for the widget.
 * @author Rowina Sanela 
 */
```
