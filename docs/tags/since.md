## <span id="syntax">语法</span>

```jsdoc
@since <versionDescription>
```
## <span id="overview">概述</span>

`@since` 标签用来表示一个类、方法或者其他标识符是在某个指定的版本加进来的。

## <span id="examples">示例</span>

```javascript
/**
 * Provides access to user information.
 * @since 1.0.1
 */
function UserRecord() {}
```
